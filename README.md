# AI-Powered CRM & Appointment Automation Platform

A production-grade CRM with AI-assisted lead management, scheduling, and follow-up automation.

## Status
🚧 Under active development — currently in Milestone 1 (project scaffolding).

## Tech Stack
- Frontend: React, TypeScript, Vite, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Database: PostgreSQL, Prisma
- AI: OpenAI API

## Structure
apps/web  – frontend
apps/api  – backend
packages/shared – shared types
docs – architecture notes
